# AuPetitVillage

Ce projet a été généré à l’aide de [Angular CLI](https://github.com/angular/angular-cli) version 19.2.24.

## Serveur de développement

Pour démarrer un serveur de développement local, exécutez :

```bash
ng serve

Une fois le serveur lancé, ouvrez votre navigateur et accédez à http://localhost:4200/. L’application se rechargera automatiquement chaque fois que vous modifiez l’un des fichiers source.

Génération de code (scaffolding)

Angular CLI inclut de puissants outils de génération de code. Pour créer un nouveau composant, exécutez :

ng generate component nom-du-composant
Pour obtenir la liste complète des schémas disponibles (tels que components, directives ou pipes), exécutez :
ng generate --help
Compilation

Pour compiler le projet, exécutez :
ng build
Cela compilera votre projet et stockera les artefacts de build dans le dossier dist/. Par défaut, la compilation de production optimise votre application pour les performances et la rapidité.

Exécution des tests unitaires

Pour exécuter les tests unitaires avec le lanceur de tests Karma
, utilisez la commande suivante :
ng test
Exécution des tests end-to-end

Pour les tests de bout en bout (e2e), exécutez :
ng e2e
Angular CLI ne fournit pas de framework de tests end-to-end par défaut. Vous pouvez en choisir un adapté à vos besoins.
